# E-Commerce Application

A Spring Boot-based e-commerce platform with MongoDB backend and JWT authentication.

## Tech Stack
- **Framework**: Spring Boot 4.0.6
- **Language**: Java 17
- **Database**: MongoDB
- **Security**: Spring Security + JWT (JJWT 0.13.0)
- **Build**: Maven

## Key Features
- RESTful APIs for e-commerce operations
- JWT-based authentication & authorization
- MongoDB data persistence
- Comprehensive test coverage

## Getting Started

### Prerequisites
- Java 17+
- MongoDB
- Maven

### Installation
```bash
git clone https://github.com/Kushagra-Sr44/E-Commerce-Application.git
cd E-Commerce-Application
mvn clean install
mvn spring-boot:run
```

### Configuration
Update `application.properties` with your MongoDB connection details.
